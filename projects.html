Programming Projects

Here’s a collection of my coding projects—some academic, some personal, and all a little chaotic in their own way.

  • Raytracer: Raytracer – A raytracing engine that parses NFF files to describe scenes and renders them with effects. Supports various object types through an enum structure including triangles, polygons, and spheres.
  • SSH-TCP: TCP project – A simulation of TCP client-server communication demonstrating handshake, reliable delivery, and packet flow logic. Also uses SSH encryption.
  • S-DES w/ Meet in the Middle Attack: S-DES – An implementation of Simplified DES along with a Meet-in-the-Middle cryptographic attack, showcasing key recovery techniques and cipher weaknesses.
  • YouTube 'Likes' Server: Likes Server – A C-based simulation that mimics a YouTube-like server using parent, primary, and likes processes (12 total). Processes communicate through sockets, log events, and validate incoming like data. Includes timeout handling and randomized like submissions via `sleep()` intervals.
