Programming Projects

Here’s a collection of my coding projects—some academic, some personal, and all a little chaotic in their own way.

Raytracer

GitHub Repository
A raytracing engine that parses NFF files to describe scenes and renders them with effects. Supports various object types through an enum structure including triangles, polygons, and spheres.

SSL-TCP

GitHub Repository
A simulation of TCP client-server communication demonstrating handshake, reliable delivery, and packet flow logic. Also uses SSL encryption.

S-DES w/ Meet in the Middle Attack

GitHub Repository
An implementation of Simplified DES along with a Meet-in-the-Middle cryptographic attack, showcasing key recovery techniques and cipher weaknesses.

YouTube 'Likes' Server

GitHub Repository
A C-based simulation that mimics a YouTube-like server using parent, primary, and likes processes (12 total). Processes communicate through sockets, log events, and validate incoming like data. Includes timeout handling and randomized like submissions via sleep() intervals.
